About Jake Durell

I first began my career in operations management and information systems. I then went on to law school seeking to understand and influence the constraints that shape solutions to world problems. As a lawyer, I drew from my technical background to help implement innovative contracting, compliance and litigation solutions. I later enrolled in Burlington Code Academy to broaden my development skills. Shortly thereafter, I began working as enterprise architect for State of Vermont at the recently formed Agency of Digital Services in the Chief Technology Office. I am also a tech lead with code for BTV, and I continue to provide representation in low and prono bono project.



I am always looking for opportunities that will embrace my development skills while leveraging my legal and business education and experience. I am gaining more experience in full stack development each day. Javascript is my core language. I have been enjoying using the Vue framework on several projects. I am also skilled in a range of Microsoft Azure products which I used to bring legacy and modern systems together, and I have been leveraging DevOps to manage these projects and their codebases. I am comfortable with a range of various project management and reporting tools, including GitHub, Trello, Visio, Sketch, Tableau, and Power BI. In addition, I have demonstrated experience in organizational and government administration, process management, database development, information systems, governance, and agile project management.

PORTFOLIO

Expunge VT

Vermont Legal Aid runs free clinics throughout the year, but copying information off state databases and filling out petitions by hand is slow. By creating a chrome extension that generates printable files in a dynamic vue JS environment at the push of a button, volunteer attorneys help more people clear their records in every clinic.

my.vermont.gov

my.vermont.gov is one piece of a single sign on initiative designed to ease access for public citizens across state agencies. Okta is used as the authentication provider. The initial version of the central landing page was built initially in vanilla JS, but a dynamic auth enable Vue JS version will launch in February 2020.

Covid Tracing & Vaccine Notifications

A rapidly developed state operated Covid solution through use of node express servers connected with on prem sql systems and cloud based texting services. The system reliably informs contacts of possible exposure, and is now being expanded to ensure citizens get timely vaccine notifications.

Where in Vermont is Ethan Allen?

Learn about Vermont geography by trying to find Ethan Allen. Uses Leaflet API for mapping, and local storage to keep track of your high scores.

Chat Amazing

A functioning parody on a nostalgic turn of the century chat service. A random name is given to each user from the randomuser.me API, and the user can then select from three chat rooms to join. The node.js server that runs the global chat service was made from scratch.

Public Records Requester

This is a beta version of a larger public records requesting app concept. This version automates the creation of a public records records request for electronic data from the Vermont State Legislature.

Nimble Health

Nimble Health is the result of a capstone project at Burlington Code Academy. It provides a web and mobile interface to facilitate non-emergency medical transportation. It uses the Google Maps API and Express.js.
