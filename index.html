About Jake Durell

I first began my career in operations management and information systems. I then went on to law school where I focused my studies on how organizations work, and the forces that govern them. As a lawyer, I drew from my technical background to help implement innovative contracting, compliance and litigation solutions. Nimble Health

Nimble Health is the result of a capstone project at Burlington Code Academy. It provides a web and mobile interface to facilitate non-emergency medical transportation. It uses the Google Maps API and Express.js.

Where in Vermont is Ethan Allen?

Learn about Vermont geography by trying to find Ethan Allen? Uses Leaflet API for mapping, and local storage to keep track of your high scores.

Chat Amazing

A functioning parody on a nostalgia inducing turn of the century chat service. A random name is given to each user from the randomuser.me API, and the user can then select from three chat rooms to join. The node.js server that runs the gloabal chat service was made from scratch.

Public Records Requester

This is a beta version of a larger public records requesting app concept. This version automates the creation of a public records records request for electronic data from the Vermont State Legislature.
